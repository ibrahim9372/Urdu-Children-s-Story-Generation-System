# Urdu Story Generator - Frontend

React + Vite frontend for the Urdu Children's Story Generation System.

## Features

- Chat-style interface for story generation
- Right-to-left (RTL) Urdu text support
- Real-time typing indicators
- Dark theme with red accent colors
- Example prompts for quick testing

## Setup

```bash
npm install
npm run dev
```

## API Configuration

The frontend connects to the backend API at `http://localhost:8000`. Update `src/utils/api.js` to change the API endpoint.

## Building

```bash
npm run build
```
